# labmu-cms
LabMu CMS: A modern, serverless CMS built natively on Cloudflare Workers. Powered by Hono, D1, and R2. Features a modular architecture with WordPress-like hooks and blazing-fast edge performance.

LabMu CMS: CMS Serverless modern yang dibangun di atas Cloudflare Workers. Ditenagai Hono, D1, dan R2. Arsitektur modular, ringan, dan super cepat di jaringan Edge.
